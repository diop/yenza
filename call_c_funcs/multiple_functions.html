Calling C functions from javascript

** C functions must be explicitly exported to be called inside Javascript**

emcc multiple_functions.c -o multiple_functions.js -s WASM=1 -O3 -s EXPORTED_FUNCTIONS="['_main', '_func1', '_func2', '_func3']"

format: var x = Module.cwrap('func-name', 'return type', ['1st argument type', '2nd arg type']);

Inside Console: func3 = Module.cwrap('func3', null, ['number'])
